# synched
================================
    Required Software
================================
    1. Node 10.20.0+
    2. Npm 6.13.4
    3. Oreos Cookies & Cream
================================
    Initialize the project
================================
    A. Server startup
        1. cd server/
        2. npm install
        3. npm start, Don't close the terminal
    B. Client startup
        1. On a new terminal type cd client/
        2. npm install
        3. npm start
